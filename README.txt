Given GEDCOM file, the program groups persons by haplogroups and prints some statistics and reports including atree. Given atree, the program generates GEDCOM file from it. Utility also can be used for text files transcoding (including into phonetic from russian windows-1251 GEDCOMs etc).

========================================================================
Installation:

Unzip aisconvert distribution (it will create "aisgedcom" directory). 

Double-click on aisgedcom-gui.bat

For the command-line version: 
 
Run -> cmd -> 
 C:\> cd aisgedcom
 C:aisgedcom> aisgedcom --help

Under Unix/Linux - run
 $ sh ./aisgedcom --help
 $ sh ./aisgedcom-gui
